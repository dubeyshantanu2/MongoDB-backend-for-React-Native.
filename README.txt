Goto to your Clusters in MongoDb Atlas and click on 
Connect-> Connect your Application-> Copy your application code-> Replace <password> with the password of your database.
Paste the application in mongoURI of Keys.js file. 

Run CMD  

You must have Node installed on your system to run NPM commands.

Install the server dependencies:
For creating backend install the following dependencies-
npm install bcrypt express jsonwebtoken mongoose body-parser

To run the server type in cmd: nodemon index

Server will be running in http://localhost:3000/

You can use POSTMAN to inspect the server activities.
